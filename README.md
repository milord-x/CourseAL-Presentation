# CourseAL-Presentation

Public presentation site for the CourseAL project.

## Live site

- GitHub Pages: https://milord-x.github.io/CourseAL-Presentation/

## Purpose

This repository hosts the public-facing presentation layer for CourseAL:

- product and concept overview
- visual pitch or demo pages
- lightweight public landing content

## Local preview

If you want to inspect the presentation locally, open the static files directly or run a small local server:

```bash
python -m http.server 8000
```

Then open `http://127.0.0.1:8000`.

## Deployment

The repository is intended to stay simple and static. Changes pushed to the default branch can be published through GitHub Pages.
